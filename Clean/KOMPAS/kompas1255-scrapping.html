https://regional.kompas.com/read/2020/10/18/23065771/diduga-korsleting-gudang-penimbunan-bbm-dan-2-rumah-warga-ludes-terbakar Diduga Korsleting, Gudang Penimbunan BBM dan 2 Rumah Warga Ludes Terbakar TUBAN, KOMPAS.com - Sebuah gudang tempat penimbunan bahan bakar minyak (BBM) jenis Solar dan dua rumah warga di Desa Leran, Kecamatan Senori, Kabupaten Tuban, Jawa Timur, hangus. Kepala Pelaksana BPBD Kabupaten Tuban, Yudi Irwanto mengatakan, peristiwa kebakaran tersebut terjadi pada Minggu (18/10/2020), sekitar pukul 16.00 WIB.  Baca juga: Istri Pertama Siswa SMK di Lombok: Waktu Itu Saya Kira Dia Tamu Mau Menjenguk, Ternyata... Api diduga berasal dari korsleting listrik pada pompa air yang terdapat di gudang tempat penimbunan BBM solar itu.  Percikan api menyambar timbunan BBM Api begitu cepat membesar dan membakar seluruh isi gudang serta menghanguskan dua rumah warga di sampingnya. "Api cepat membesar lantaran menyambar minyak solar," kata Yudi saat dihubungi Kompas.com, Minggu (18/10/2020). Warga sekitar yang membantu memadamkan api dengan peralatan seadanya tak bisa berbuat banyak Api membesar dan membakar gudang yang belum diketahui pemiliknya itu. Sebanyak tiga mobil pemadam kebakaran dan satu mobil tangki air turun ke lapangan menjinakkan api. “Pemadaman api juga dibantu satu mobil Damkar Bojonegoro dan satu mobil Damkar Pertamina Cepu Api berhasil dipadamkan pada pukul 19.04 WIB,” terangnya. Yudi menjelaskan, dua rumah yang habis terbakar merupakan milik warga bernama Jasmini (70) dan Kasman (65).
