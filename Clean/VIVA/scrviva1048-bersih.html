https://www.viva.co.id/berita/metro/1311063-api-dari-lapak-kusen-hanguskan-sejumlah-bangunan-di-depok Api dari Lapak Kusen Hanguskan Sejumlah Bangunan di Depok VIVA – Kebakaran besar terjadi di Jalan Tole Iskandar, Kecamatan Sukmajaya, Depok, Jawa Barat Kobaran api menghanguskan sejumlah bangunan yang ada di kawasan tersebut pada Minggu malam 11 Oktober 2020.  Kapolsek Sukmajaya, Ajun Komisaris Polisi Ibrahim Sadjab menuturkan, api diduga berasal dari lapak kusen yang berada di dekat warung makan Tegal atau warteg, sekira pukul 18:30 WIB.  "Saat itu saksi melihat ada asap tebal dan api di pembakaran sisa-sisa kayu Lalu saksi berusaha untuk minta tolong warga setempat," kata Ibrahim di lokasi kejadian. Baca Juga: Kebakaran Bioskop di Senen, 25 Unit Mobil Damkar Diterjunkan Pun, karena banyaknya material yang mudah terbakar, api pun dengan cepat merambat hingga akhirnya menghanguskan sejumlah bangunan yang ada di sekitar lokasi kejadian.          "Tidak ada korban jiwa, saat ini kasusnya sedang kami dalami," ujarnya. Sementara itu, Komandan Pleton Operasional Pemadaman dan Penyelamatan Dinas Damkar Kota Depok, Merdy Setiawan mengatakan, pihaknya masih berupaya memadamkan api.  Di tempat yang sama, Wakil Wali Kota Depok definitif, Pradi Supriatna yang datang langsung ke lokasi kejadian mengatakan, dirinya akan berupaya membantu korban terdampak kebakaran.  "Minimal kita akan bantu cari tempat untuk menginap sementara, ini akan kami cari datanya ke RT atau RW," ujar Pradi Selanjutnya, ia juga berjanji akan berkoordinasi dengan pejabat sementara (Pjs) Wali Kota Depok, Dedi Supandi guna penanganan lebih lanjut Hingga pukul 21.05 wib, api masih masih berkobar hingga menyebabkan kemacetan.  Belum diketahui kerugian akibat kebakaran ini.
