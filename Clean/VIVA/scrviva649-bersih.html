https://www.viva.co.id/berita/kriminal/1308603-terungkap-penyebab-koboi-jalanan-tenteng-senjata-api-di-bekasi Terungkap, Penyebab Koboi Jalanan Tenteng Senjata Api di Bekasi VIVA – Polisi telah mencokok pemuda berinisial EA (38) yang menenteng senjata api laiknya koboi jalanan di kawasan Bekasi Usut punya usut, tindakannya itu dilakukan karena dia ribut dengan seorang juru parkir. "Ribut sama tukang parkir," ucap Kapolsek Tarumajaya, Ajun Komisaris Polisi Yudho Anto Hutri kepada wartawan, Minggu 4 Oktober 2020. Semua berawal ketika pelaku tengah memundurkan mobilnya Kemudian si jukir membantu memundurkannya dengan memberi arahan Sayangnya, saat memundurkan mobil pelaku malah mengenai si jukir Alhasil si jukir menegur pelaku, tapi pelaku kesal. Baca juga: Gibran Tambah Virtual Box untuk Kampanye, Megawati pun Bisa Pakai "Masalah parkiran saja Jadi, dia mundur (mengendarai mobil), dimundurin tukang parkir, terus mobilnya malah kena (hantam) tukang parkir Ditegur (sama tukang parkirnya, marah-marah gitu, cekcok mulut," katanya. Sebelumnya diberitakan, video seorang pemuda menenteng senjata api layaknya seorang koboi di Kampung Bogor, Tarumajaya, Kabupaten Bekasi viral di media sosial Pemicunya hanya kesalahpahaman dengan juru parkir. Dalam postingan disebut kejadian ini terjadi Sabtu 3 Oktober 2020 kemarin Kejadian tersebut dikatakan terjadi di Kawasan Bekasi sekira pukul 17.00 WIB sore Dalam video tersebut memperlihatkan seorang pria bertubuh gempal menenteng senjata api saat terlibat salah paham dengan juru parkir. Video yang berdurasi 43 detik itu ikut diposting akun instagram bekasi.terkini Pria itu menghampiri juru parkir sambil menenteng senjata api "Kamu jangan macem-macem," kata pria itu. "Saya cuma parkir," saut juru parkir itu "Monyet," tambah pria yang mengendarai Avanza hitam bernomor polisi B-1537-TMO Peristiwa itu pun langsung viral.
