https://www.viva.co.id/berita/metro/1299679-lapak-pedagang-kelapa-di-juanda-depok-kebakaran Lapak Pedagang Kelapa di Juanda Depok Kebakaran VIVA – Api menghanguskan bangunan liar yang menjadi lapak pedagang kelapa di Gang Saminten, Jalan Juanda Depok, Minggu malam, 6 September 2020 Berdasarkan informasi yang diterima petugas, kebakaran terjadi sekira pukul 18.30 WIB. Baca: Kebakaran Bedeng di Depan Asrama Mabesad Gambir, 13 Unit Damkar Turun Berdasarkan video yang beredar di media sosial, api dengan cepat menjalar dan membumbung tinggi, sehingga menarik perhatian warga sekitar Tak berapa lama mobil petugas dinas pemadam kebakaran Kota Depok segera tiba di lokasi dan melakukan pemadaman. Bangunan yang terbuat dari kayu membuat api cepat membesar dan menghanguskan seluruh bangunan lapak Lalu lintas di Jalan Juanda Depok sempat tersendat imbas dari kebakaran. Dinas Penyelamatan dan Pemadam Kebakaran (DPPK) Kota Depok berhasil memadamkan api kurang dari satu jam Api berhasil dipadamkan dan mencegah api menjalan ke pemukiman warga yang ada di depan lapak yang terbakar. Menurut informasi petugas, api berasal dari bara api yang masih menyala di dalam tumpukan kayu kering sehingga mengeluarkan api, membakar dan menghanguskan lapak   Beruntung tidak ada korban jiwa dalam insiden tersebut Namun demikian, lapak di sekitar lokasi seluruhnya hangus terbakar.
