https://www.viva.co.id/berita/metro/1291642-kebakaran-pabrik-mebel-di-cakung-petugas-masih-kesulitan-padamkan-api Kebakaran Pabrik Mebel di Cakung, Petugas Masih Kesulitan Padamkan Api VIVA – Kebakaran hebat terjadi di sebuah pabrik mebel yang terletak di Jalan Raya Penggilingan, Cakung, Jakarta Timur pada Minggu, 9 Agustus 2020 Hingga saat ini, petugas dari pemadam kebakaran Jakarta Timur masih berupaya memadamkan api. Banyaknya bahan yang mudah terbakar di dalam pabrik mebel membuat api sulit dipadamkan Suku Dinas Penanggulangan Kebakaran dan Penyelamatan (Sudin PKP) Jakarta Timur awalnya menerima laporan kebakaran sebuah pabrik mebel itu sekitar pukul 12.02 WIB. Namun, sudah empat jam berselang, kobaran api dan kepulan asap tebal masih terlihat di lokasi Lamanya proses pemadaman, lantaran sulitnya sumber air. Baca juga: Pabrik Mebel di Cakung Terbakar Hebat, Api Belum Padam "Kendalanya sumber air sangat jauh Kami terus mengupayakan menambahkan unit untuk membantu sumber air sampai ke TKP," kata Kasie Ops Sudin Gulkarmat Jakarta Timur, Gatot Sulaeman, di lokasi. Pantauan VIVA di lokasi, sejumlah warga bahu-membahu di lokasi untuk memadamkan kobaran api Sementara itu, beberapa dari warga turut membantu mengatur lalu lintas untuk memudahkan unit damkar menuju TKP. "Pabrik mebel yang terbakar adalah kayu produksi dari mebel itu sendiri Asal api, salah satu karyawan mengatakan berasal dari pembuatan meja, ada ledakan dan merambat ke sekitarnya," sambung Gatot. Selain dari unit mobil pemadam kebakaran dari Sudin Penanggulangan Kebakaran dan Penyelamatan Jakarta Timur, terlihat juga empat unit mobil pemadam kebakaran dari Sudin Penanggulangan Kebakaran dan Penyelamatan Kota Bekasi Hingga berita ini ditulis, kobaran api masih terlihat dari gedung pabrik mebel (art)
