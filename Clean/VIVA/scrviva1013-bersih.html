https://www.viva.co.id/berita/metro/1309010-rumah-mewah-di-menteng-terbakar-sumber-api-dari-mini-cooper Rumah Mewah di Menteng Terbakar, Sumber Api dari Mini Cooper VIVA – Sebuah rumah tinggal di Jalan Lembang, Kecamatan Menteng, Jakarta Pusat hangus dilalap api Peristiwa kebakaran yang terjadi Senin siang itu diduga berasal dari mesin mobil dan saat ini api telah dipadamkan. Kepala Sudin Gulkarmat Jakarta Pusat, Asril Rizal, mengatakan, api sudah berhasil dipadamkan dengan menerjunkan enam unit mobil pemadam kebakaran Kebakaran tersebut berasal dari mesin mobil Mini Cooper yang berada di garasi rumah. Baca juga: Pengedar Sabu Kelas Kakap Sembunyi di Apartemen Kawasan Cipinang “Info awal kebakaran terjadi dari mobil Mini Cooper warna hijau dengan nomor polisi B 1 JCW milik Franc Susanto Mobil itu berada di dalam garasi rumah Karena api tidak terkendali, sehingga membesar dan merambat ke sekitar area garasi," kata Asri dalam keterangan tertulis, Senin 5 Oktober 2020. Asril menambahkan, dalam kebakaran tersebut tidak ada korban jiwa Namun, kerugian ditaksir mencapai Rp1 miliar “Taksiran kerugian Rp1.010.000.000 Korban nihil Itu aja,” tuturnya.
