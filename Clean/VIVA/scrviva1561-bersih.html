https://www.viva.co.id/berita/nasional/1294325-pabrik-triplek-di-jambi-terbakar-karyawan-berhamburan Pabrik Triplek di Jambi Terbakar, Karyawan Berhamburan VIVA - Kebakaran besar terjadi di dalam pabrik triplek RT 10, Desa Sungai Duren, Kecamatan Jaluko, Kabupaten Muaro Jambi, Selasa, 18 Agustus 2020 Tempat pengeringan triplek yang berbahan baku kayu dan gudang boiler hangus terbakar. Informasi yang dihimpun VIVA, kebakaran pabrik triplek diketahui saat jam istirahat sekitar pukul 12.40 WIB Sementara para pekerja yang mengetahui kebakaran langsung berhamburan keluar pabrik dan sebagian karyawan lainnya membantu memadamkan api. Baca juga: Rumah Warga Rusak Akibat Pabrik Bioethanol Terbakar hingga Meledak Kapolres Muaro Jambi, AKBP Ardiyanto, membenarkan adanya kebakaran pabrik triplek tepatnya bagian gudang Tempat Pengeringan Triplek dan gudang Boiler PT Duren Mandiri Fortune, Desa Sungai Duren, Kabupaten Muaro Jambi. "Ya benar ada, kebakaran pabrik triplek dan sangat besar kebakarannya dan api baru bisa dipadamkan sekitar pukul 14.00 WIB," ujarnya. Ardiyanto menceritakan saat kebakaran di pabrik, anggota Polsek Jaluko langsung turun ke lokasi kebakaran Saat di lokasi, masyarakat lokasi desa, karyawan pabrik, pemadam kebakaran dan kepolisian berjibaku memadamkan api Dalam kejadian itu tidak ada korban jiwa. "Kalau kronologi kebakaran, saat karyawan sedang istirahat makan siang dan saat itu posisi gudang dalam keadaan kosong ditinggal pekerja, namun para karyawan terkejut api sudah membesar di bagian tempat pengeringan triplek yang berbahan baku kayu," katanya. Terpisah, Kapolsek Jaluko, Iptu Irwan, membenarkan ada kebakaran di pabrik triplek dan beruntung tidak ada korban jiwa Pemadaman dilakukan oleh petugas gabungan dari Polsek, BPBD dan Damkar. "Saat saya dapat informasi, tim anggota langsung turun ke lokasi kebakaran pabrik triplek dan saat api besar, anggota dan saya sebagai kapolsek ikut membantu memadamkan api dan kurang lebih dua jam, api baru bisa dijinakkan," katanya. Irwan mengatakan untuk kerugian materiil sementara belum bisa ditaksir Sampai saat ini, mereka masih menyelidiki kejadian tersebut.
