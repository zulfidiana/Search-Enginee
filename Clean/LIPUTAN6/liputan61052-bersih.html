https://www.liputan6.com/news/read/4387003/pemprov-dki-klaim-peminat-hunian-dp-0-rupiah-meningkat-saat-pandemi Pemprov DKI Klaim Peminat Hunian DP 0 Rupiah Meningkat Saat Pandemi - News Liputan6.com Liputan6.com, Jakarta Direktur Utama PD Sarana Jaya, Yoory C Pinontoan menyatakan pandemi virus corona atau Covid-19 tidak menyurutkan peminat hunian DP 0 rupiah di Jakarta. Dia mengatakan berdasarkan data yang ada puluhan ribuan orang telah mendaftarkan diri untuk membeli rumah dari program Pemprov DKI. Baca Juga "Kalau melihat datanya kan besar, pendaftaran kan sampai dengan 21 ribu, itu pendaftaran yah kurang lebih," kata Yoory dalam keterangan tertulis, Selasa (20/10/2020). Kendati begitu, dia mengatakan pihaknya akan bekerjasama dengan sejumlah pihak untuk meningkatkan hunian DP 0 rupiah di Ibu Kota Sebab akibat pandemi Covid-19, sejumlah proyek pembangunan terhambat.
